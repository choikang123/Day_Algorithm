이렇게 하면 내 예상은 
["but", "wont", "it"]
이런식으로 들어갈 줄 알았는데 list()의 중요한 특징이 문자열이 들어오면 알파벳별로 다 쪼개서 리스트로 만들어주기때문에
["b","u","t] 이렇게 들어가서 문제가 될 수 있음
그리고 중요한게 문자열을 입력 받을때는 .strip() 을 써주어야지 뒤에 "\n"을 없애주기 때문에 항상 지워줄것
